We appreciate contributions in all forms! By contributing to Mentor, you'll help make it a better tool for the RDF and knowledge graph community. Contributions can take many shapes, including:

### Bug Reports
If you encounter an issue, please report it to us so we can investigate and fix it. Your feedback helps us understand what works well and what we can improve.

### Questions & Answers (Q&A)
If you have questions or need assistance, feel free to ask in the [Mentor Discussions](https://github.com/faubulous/mentor-vscode/discussions) forum.

### Reviews and Feedback
Share your experience with Mentor by leaving a review on the [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=faubulous.mentor).

### Code Contributions
Help us improve the extension by submitting new features, bug fixes, or refactoring existing code.
