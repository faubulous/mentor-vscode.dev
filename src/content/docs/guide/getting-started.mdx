Mentor is an extension for Visual Studio Code. This means that you will need to install Visual Studio Code or one of its derivatives.

<img class="screenshot my-3" src="/screenshots/extensions.png" alt="Screenshot of the Extensions panel in Visual Studio Code" width="300" />

To get started with Mentor, follow these steps:

<Steps>
1. Install [Visual Studio Code](https://code.visualstudio.com), [VSCodium](https://vscodium.com/) or go to [vscode.dev](https://vscode.dev)
2. Install the [Mentor Extension](https://marketplace.visualstudio.com/items?itemName=faubulous.mentor):

   Open Visual Studio Code, go to the Extensions view by clicking on the Extensions icon in the Activity Bar on the side of the window or by pressing `Ctrl+Shift+X`. Search for "Mentor" and click "Install" to add the extension to your IDE.

3. Open or Create a Workspace
</Steps>
