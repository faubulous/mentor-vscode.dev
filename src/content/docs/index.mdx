## News
<Aside icon="rss" title="Version 0.5.0 - 16th March 2026">
Official release of Mentor with support for RDF 1.2 syntax in N-Triples, N-Quads, Turtle, TriG, 
N3 based on the new [mentor-rdf-parsers](https://github.com/faubulous/mentor-rdf-parsers) library.

*New Features:*
- Improved workspace index speed up to x4
- Full RDF 1.2 support for N-Triples, N-Quads, Turtle, TriG, N3 and SPARQL
  - Code formatting
  - Format conversion
- Added support for toggling inference in SPARQL queries
  - Workspace inference graphs are hidden by default
- Internal refactoring
  - Faster extension activation and smaller package size
  - No more CORS restrictions when running in VSCode desktop
</Aside>
