[Mentor](https://marketplace.visualstudio.com/items?itemName=faubulous.mentor) is an extension for Visual Studio Code. This means that you will need to install [Visual Studio Code](https://code.visualstudio.com), [VSCodium](https://vscodium.com/) or in the browser by going to [vscode.dev](https://vscode.dev).

To get started with Mentor, follow these steps:

<Steps>
1. Go to the Extensions view by clicking on the Extensions icon in the Activity Bar on the side of the window or by pressing `Ctrl+Shift+X` (Windows, Linux) or `Cmd+Shift+X` (macOS).
2. Search for "Mentor" and click "Install" to add the extension to your IDE.
3. Open or Create a Workspace.
</Steps>
